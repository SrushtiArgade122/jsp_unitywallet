# 💳 JSP UnityWallet

**JSP UnityWallet** is a production-ready mobile financial portal developed as a **Community Engagement Project (CEP)** and professional consultancy for the *Junnar Taluka Prathmik Shikshak Sahakari Pat Sanstha*. 

The application provides over **1,000 teacher-members** with real-time, secure access to their financial records, eliminating the need for physical office visits.

---

## 🚀 Project Highlights
* **Zero-Cost Hosting:** Leverages a serverless architecture to ensure the society pays ₹0 in monthly maintenance or hosting fees.
* **Real-time Sync:** Instant synchronization between Google Sheets (Administrative Backend) and the Flutter UI.
* **Professional Delivery:** Successfully delivered as a paid consultancy for a fee of **₹3,000**.

---

## 🛠 Tech Stack
* **Frontend:** [Flutter](https://flutter.dev) (Dart) - For a responsive, cross-platform mobile experience.
* **Backend/Database:** [Google Sheets](https://www.google.com/sheets/about/) - Used as a cloud-hosted relational data store.
* **Middleware/API:** [Google Apps Script](https://www.google.com/script/start/) - Functions as a RESTful API bridge using JSON.

---

## 📱 Key Features
- **Secure Member Login:** Authentication using unique Member IDs.
- **Financial Dashboard:** "Warm Modern" UI displaying:
    - Loan Principal & Interest Balances
    - Death Fund (ठेव) Contributions
    - Total Dividends/Savings
- **Privacy First:** Data is filtered at the server level so members only see their own records.

---

## 🏗 System Architecture
The app follows a three-tier serverless architecture:
1. **Presentation Layer:** Flutter Mobile App.
2. **Logic Layer:** Google Apps Script (REST API).
3. **Data Layer:** Google Sheets.
